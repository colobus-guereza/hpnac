[임시 비활성화된 기능 목록]

1. 스케일 사전 기능
   - 위치: script.js
   - 상태: 주석 처리됨
   - 관련 코드:
     * 라우팅: '/scale-dictionary': scaleDictionaryView
     * 교육자료 메뉴의 스케일 사전 버튼
   - 구현된 기능:
     * D Kurd 10 스케일 정보 표시
     * 스케일 특징 목록
     * YouTube 비디오 연동

2. 강사 연락처 기능
   - 위치: script.js (teachersView 함수 내)
   - 상태: HTML 주석 처리됨
   - 관련 코드: 각 강사 카드의 전화연결 버튼
   - 구현된 전화번호:
     * 이헌국: +821089679204
     * 김문겸: +821029388815
     * 미지수: +821045299038
     * 안재민: +821072297450
     * 이시온: +821044454689
     * 이지은: +821045299038
     * 조진희: +821031270691

3. 카카오톡 본인인증 세부 기능
   - 위치: script.js
   - 상태: 더미 데이터로 구현됨
   - 관련 코드:
     * YOUR_KAKAO_APP_KEY
     * YOUR_REDIRECT_URI
     * 테스트용 폼 URL

4. 강사 프로필 카드 노출 현황
   - 위치: script.js (teachersView 함수 내)
   - 활성화된 강사:
     * 이헌국 (원장)
     * 이시온 (명상 음악가)
     * 이지은 (음악치료, 명상상담)
   - 비활성화된 강사:
     * 김문겸 (국악 연주자)
     * 미지수 (배우 & 연주자)
     * 안재민 (배우 & 연주자)
     * 조진희 (암환우 연주자)
   - 상태: 코드 제거됨 (추후 필요시 재추가 예정)

5. 필기시험 접수 기능
   - 위치: script.js (certificationView 함수 내)
   - 상태: 버튼 및 관련 코드 제거됨
   - 관련 코드: 
     * 자격증 카드의 '필기시험 접수' 버튼
     * onclick="navigateTo('/auth')" 이벤트 핸들러
   - 본인인증 페이지 URL: https://hpnac.kr/auth/certification
   - 비고: 시험 응시 희망자에게 개별적으로 URL 전달 예정
